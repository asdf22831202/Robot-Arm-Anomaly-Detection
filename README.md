# 機器手臂異常監測

本專案利用三軸振動訊號建立機械手臂傳動系統之健康監測模型，並以 Streamlit 建立可視化介面，實現不停機之異常偵測。

原始資料僅供學術用途，基於資料擁有權限制，本 repo 不公開原始振動資料。
資料格式請見 Data/README.md。

---

## 專案核心目標

1. **預防突發故障**：在設備明顯失效前提供預警，降低非計畫性停機風險  
2. **優化感測器配置**：比較同方向不同感測器位置的模型表現，挑選最佳量測點  
3. **簡化數據分析**：以儀表板/指示燈方式輸出結果，使非專業人員也能快速判讀

---

## 方法架構

### CNN：頻域風險建模

- 使用 Welch 方法計算 PSD / CSD
- 建立 3×3 類共變異矩陣頻域張量
  |        | x-axis | y-axis | z-axis |
  |--------|--------|--------|--------|
  | **x**  | Pxx    | Pxy    | Pxz    |
  | **y**  | Pyx    | Pyy    | Pyz    |
  | **z**  | Pzx    | Pzy    | Pzz    |
- 進行離散型（分類）與連續型（迴歸）建模
- 使用 Stratified 5-fold 驗證穩定性

### Random Forest：健康風險分層決策模型

#### 三階段特徵篩選

1. Filter：mRMR / FCBF / CMIM / JMI / ReliefF
2. Wrapper：Conditional Permutation Importance (CPI)
3. SHAP 分層篩選與 LOO 驗證

#### 兩層決策架構

第一層：Mean Model  
- 輸出健康機率（HI）
- 以健康 1% 分位數與異常 99% 分位數設為灰色地帶

第二層：負荷子模型  
- 若任一子模型判定異常 → 視為異常
- 若落入灰色地帶 ≥ 2 次 → 視為異常

---

## 感測器選擇結果

- X 方向：選擇馬達側（Xa）：較低且穩定 MSE
- Y 方向：選擇馬達側（Ya）：分類表現一致性較佳

---

## 模型特性比較

| 模型 | 優點 | 限制 |
|------|------|------|
| CNN | 預測穩定、誤差較低 | 需固定長度（X≥20000, Y≥21000） |
| RF  | 無長度限制、可解釋性高 | 推論較慢、整體準確率略低 |
